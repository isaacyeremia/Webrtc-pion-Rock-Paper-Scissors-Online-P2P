# Webrtc-pion-Rock-Paper-Scissors-Online-P2P

aplikasi game Rock Paper Scissors multiplayer berbasis web yang mengimplementasikan teknologi WebRTC DataChannel. Aplikasi memungkinkan dua pemain untuk bermain secara real-time melalui browser, dengan server Golang yang berperan sebagai matchmaking dan relay data menggunakan library Pion WebRTC.

cara menjalankannya pertama harus menjalankan golangnya dulu, lalu untuk menjalankan htmlnya untuk saat ini saya buatnya harus menjalankan dengan extensions vscode yaitu go live
